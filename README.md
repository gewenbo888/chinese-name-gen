# Chinese Name Generator — 中英文取名器

A bilingual name generator with cultural analysis.

[![Live Demo](https://img.shields.io/badge/demo-chinese--name--gen.vercel.app-a8322c)](https://chinese-name-gen.vercel.app)

## Overview

Generate beautiful Chinese and English names with cultural context. Choose style, gender, and origin; each result comes with pinyin, character meaning, and a short analysis so you can pick a name that actually means something.

## Design & Tech

- **Stack:** Single HTML file, vanilla JavaScript
- **Aesthetic:** Editorial Chinese-English parallel layout
- **Deployment:** Vercel
